📡 Campus Network Monitor
A lightweight, real-time Network Monitoring Dashboard built with Python and Flask. This tool allows you to track the connectivity of critical network infrastructure (Routers, PCs, Servers) through a clean, auto-refreshing web interface.

✨ Features
Real-time Status: Automatically pings a list of devices to check for connectivity.

Auto-Refresh: The dashboard updates every 5 seconds to provide the latest network state.

Cross-Platform: Smart detection for ping commands on both Windows and Linux systems.

Clean UI: Simple, color-coded status indicators (🟢 UP / 🔴 DOWN).

Highly Customizable: Easily add or remove devices via a simple Python dictionary.

🛠️ Tech Stack
Backend: Python 3.x

Web Framework: Flask

Networking: ICMP (Subprocess)

Frontend: Jinja2 Templates & CSS3
